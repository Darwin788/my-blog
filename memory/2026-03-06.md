- 子模块没有正确配置 .gitmodules
- Jekyll 构建时检测到嵌套仓库导致冲突

1. `git rm --cached resume` 移除子模块跟踪
2. `rm -rf resume/.git` 删除嵌套的 .git 目录
3. 重新提交推送，触发构建
