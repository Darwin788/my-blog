如何用 OpenClaw 搭建智能助手

AI 助手已经成为工作和生活的标配，但你是否想过搭建一个属于自己的智能助手？

OpenClaw 是一个开源的 AI 助手框架，帮你快速构建个性化 AI 助手，实现自动化工作流。

一、什么是 OpenClaw？

OpenClaw 是一个开源的 AI 助手框架，特点：

特性说明
开源免费GitHub 开源，无隐藏费用
多渠道支持支持 QQ、Telegram、微信、Discord 等
技能市场可安装第三方技能，也可自己开发
高度定制可自定义提示词、模型、工作流
本地部署数据保存在本地，隐私安全

二、安装与环境准备

步骤 1：安装 Node.js

OpenClaw 基于 Node.js 开发，需要先安装 Node.js 环境。

官网下载：https://nodejs.org/ （建议 v18 以上版本）

验证安装：node -v

步骤 2：安装 OpenClaw

npm install -g openclaw

步骤 3：初始化项目

openclaw init my-assistant
cd my-assistant
npm install

步骤 4：配置模型

修改 config/default.json，配置你的 API Key：

{
  "model": {
    "provider": "qianfan",
    "apiKey": "your-api-key-here"
  }
}

三、创建第一个技能

OpenClaw 的核心是「技能(Skill)」，一个技能可以完成特定任务。

创建技能文件

在 skills/ 目录下创建新技能：

├── skills/
│   └── my-first-skill/
│       ├── SKILL.md      # 技能说明
│       └── handler.js    # 技能逻辑

编写技能逻辑

// handler.js
module.exports = async (context) => {
  const { message, args } = context;
  
  // 处理用户输入
  const userInput = args.join(' ');
  
  // 返回响应
  return `你刚才说的是：${userInput}`;
};

编写技能说明

# 我的第一个技能

## 功能
echo 用户输入的内容

## 使用方式
!echo [内容]

## 示例
!echo Hello World

四、配置聊天渠道

OpenClaw 支持多种聊天渠道，以下是 QQ 渠道的配置：

安装 QQ 适配器

openclaw install qqbot

配置 QQ 登录

修改 config/qqbot.json：

{
  "protocol": "平板",
  "deviceInfo": {
    "product": "MI 6",
    "model": "sagit",
    "androidVersion": 29
  }
}

五、高级功能：技能市场

OpenClaw 有技能市场，可以直接安装现成的技能：

# 查看可用技能
openclaw search [关键词]

# 安装技能
openclaw install [技能名]

# 查看已安装技能
openclaw list

常用技能推荐

技能名功能
weather查询天气
qqbot-cron定时提醒
agent-browser浏览器自动化
proactive-agent-lite主动式 AI 助手

六、实战案例：医美文案助手

下面演示如何创建一个「医美文案助手」技能：

创建技能目录

mkdir -p skills/kos-copywriter
cd skills/kos-copywriter

编写 SKILL.md

# 医美文案助手

## 功能
- 生成植发相关文案
- 优化现有文案
- 卖点转买点

## 使用方式
!copy 主题
!optimize 原文案

编写 handler.js

module.exports = async (context) => {
  const { args, model } = context;
  
  if (args[0] === 'copy') {
    // 生成文案
    const topic = args.slice(1).join(' ');
    return await model.complete(`请为以下医美主题写一篇小红书风格的文案：${topic}`);
  }
  
  if (args[0] === 'optimize') {
    // 优化文案
    const content = args.slice(1).join(' ');
    return await model.complete(`请将以下医美文案进行卖点转买点优化：\n${content}`);
  }
  
  return '用法：!copy 主题 或 !optimize 原文案';
};

写在最后

OpenClaw 的强大之处在于可扩展性。你可以通过技能市场获取功能，也可以自己开发定制化技能。

更重要的是，OpenClaw 支持本地部署，数据完全掌握在自己手中。
